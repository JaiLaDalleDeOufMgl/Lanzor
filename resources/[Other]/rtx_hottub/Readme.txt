Install instructions:
1. Rename rtx_hottub_object.pack to rtx_hottub_object
2. Put rtx_hottub_object folder to your resources
3. Put rtx_hottub_object to the server.cfg
4. Rename rtx_hottub.pack to rtx_hottub
5. Put rtx_hottub folder to your resources
6. Configure your config
7. Put rtx_hottub to the server.cfg
8. Upload sql.sql file to your mysql database.
